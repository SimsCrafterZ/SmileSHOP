What is it?<br>
-It's an unistore for installing SmileBASIC programs directly onto your 3DS.<br><br>
how to install it?<br>
-It's easy! Scan this QR code with Universal Updater.<br>
<img width="360" height="360" alt="1000102914" src="https://github.com/user-attachments/assets/2aaef5d5-4c33-4a6d-9b7a-5c4503711d52" /><br><br>
How to use it?<br>
It's easy! Install the program you want and you will get a T"program name" file (and sometimes, a BBG"program name" file and/or a BSP"program name" file) in the SmileSHOP folder, example: sd/SmileSHOP/TMARIO, sd/SmileSHOP/BBGMARIO, sd/SmileSHOP/BSPMARIO.<br>
Then move all these files into the SmileBASIC folder using sbfm (SmileBASIC file manager), available here: https://github.com/nnn1590/lpp-3ds-sbfm
